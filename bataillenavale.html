Projet de première année : la Bataille Navale

Mon premier projet en PHP, le but était de développer une bataille navale dans le terminal en faisant appel à des tableaux a deux dimensions. J'ai rajouté un code qui analyse le tableau et empêche d'empiler les bateaux mais aussi de poser un bateau en dehors du tableau.

Ensuite j'ai créé un deuxième tableau qui servirait de mémoire pour les endroit où un joueur tire, il conserve en mémoire si un bateau a été touché à cet endroit, dans ce cas le second tableau affiche un 1, et il garde aussi en mémoire les endroit où le joueur tire mais ne trouve pas de cible dans ce cas là le tableau affiche un 2 à cet endroit.

Voici dessous un tableau montrant un bateau touché. On remarque que le code nous envoie un message nous rappelant combien de cible il nous reste à toucher, cela fonctionne grâce a un compteur qui se remet à jour à chaque fois qu'un bateau est touché.
