Projet de première année : Base de Données de Gîtes

Le but de ce projet a été de modélisé une base de données avec la méthode Meurise pour simplifier l’organisation d’un site de gestion de divers gîtes.

Nous avons tout d’abord défini quels étaient les données importantes et à quelle entitée elles appartenaient.

Nous avons ensuite créé un schéma qui permettait de voir les associations entre les différents éléments de notre base de données.

Ce qui nous a permis de créé les tables de la base de données en suivant le modèle si dessous.

Enfin nous avons créé les instances des éléments comme les informations de chaque gîte.
